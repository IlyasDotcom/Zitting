# KvK Bedrijventeller – Afstudeerverdediging

Interactieve presentatie-app voor de afstudeerverdediging van Ilyas Malik.

## Starten

```bash
npm install
npm run dev
```

Open http://localhost:5173 in je browser.

## Navigatie

- **→ / ←** of knoppen onderaan: volgende/vorige slide
- **N**: speaker notes aan/uit
- Klik op secties in de navigatiebalk om direct te springen
